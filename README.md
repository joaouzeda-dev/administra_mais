# administra_mais
Administra+ 🚀 | Sistema de gestão administrativa e financeira para controle de alunos e fluxo de caixa. Desenvolvido com React, TypeScript e Supabase, utilizando a plataforma de IA Lovable para prototipagem ágil e interface moderna com Tailwind CSS. Foco em performance, segurança e usabilidade intuitiva.
