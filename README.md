# 🚀 Administra+

O **Administra+** é uma aplicação completa para gerenciamento administrativo e financeiro. Focado na facilidade de uso, o sistema permite o controle centralizado de receitas, despesas e assinaturas de alunos em uma interface moderna e responsiva.

> ⚡ **Desenvolvimento:** Este projeto foi concebido e estruturado utilizando a plataforma **Lovable**, integrando IA generativa para acelerar a construção da UI e a lógica inicial do sistema.

## 🛠️ Tecnologias
* **Frontend:** React.js & TypeScript
* **Estilização:** Tailwind CSS & shadcn/ui
* **Backend:** Supabase (Auth & PostgreSQL)
* **Build:** Vite
* **Ícones:** Lucide React

## ✨ Funcionalidades
* 📊 **Dashboard:** Visão geral de receitas e despesas em tempo real.
* 👥 **Gestão de Alunos:** Controle de assinaturas e dados cadastrais.
* 💰 **Financeiro:** Fluxo de caixa detalhado com histórico de transações.
* 🔐 **Segurança:** Autenticação robusta e proteção de dados via Supabase.
